Current State:

Built a plotting drawing robot that can move to any point on the canvas in a straight line. Developed software that can take an input image and output a coninous array of points to draw the image. The image processing software uses Canny edge detection (OpenCV) and Random Forest classifier (Scikit) to filter edges and make one continuos line to draw the image.
